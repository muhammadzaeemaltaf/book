# Appendices

## Additional Resources

This section contains supplementary materials to support your learning journey through the Physical AI & Humanoid Robotics textbook.

### Available Appendices

- **Hardware Setup Guide**: Requirements and configuration for different hardware tiers
- **Software Installation Guide**: Step-by-step setup for Ubuntu 22.04 + ROS 2 Humble
- **Troubleshooting Guide**: Common errors and solutions
- **Resources and Further Learning**: Links to official documentation and advanced materials

### System Requirements

For the best learning experience, we recommend the following specifications:

- **Minimum**: 8GB RAM, 20GB free storage, Ubuntu 22.04 LTS
- **Recommended**: 16GB+ RAM, 50GB+ free storage, dedicated GPU for Isaac Sim
- **High-end**: 32GB+ RAM, SSD storage, RTX-class GPU for advanced simulation
